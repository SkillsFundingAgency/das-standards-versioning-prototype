What does it mean to opt in or apply for a standard version?

You can opt into a new standard version if you've already been approved for a different version with the same assessment plan. This means you'll be automatically approved and won't have to answer questions about your organisation's assessment capabilities and capacity.

You can apply for a new standard version if you've already been approved for a different version of the same standard. This means you won't have to answer as many questions to get approved.
